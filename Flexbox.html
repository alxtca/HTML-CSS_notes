The flex container properties are:

  • flex-direction
  • flex-wrap
  • flex-flow : shorthand property for setting both the flex-direction and flex-wrap properties (flex-flow: row wrap;)
  • justify-content : is used to align the flex items (on X-axis ---center--->) (flex-start, flex-end, center, space-around, space-between)
  • align-items : is used to align the flex items (on Y-axis) (flex-start, flex-end, center, stretch, baseline) - when flex-direction: column;
  • align-content : is used to align the flex lines (flex-start, center, stretch, space-around, space-between)

The flex children properties are:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex : is a shorthand property for the flex-grow, flex-shrink, and flex-basis properties (flex: 0 0 200px)
  • align-self: is used to align idividual items (flex-start, flex-end, center, stretch, baseline)
